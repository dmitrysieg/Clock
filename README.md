# Clock
Arduino code for Grow Light Timer.
Using Troyka modules:
- Display
- RTC
- Relay 220V

Using libraries:
- QuadDisplay2 (https://github.com/amperka/QuadDisplay2)
- TroykaRTC (https://github.com/amperka/TroykaRTC)
